# Ruvishan Maleesha - Portfolio

Welcome to my personal portfolio showcasing my work, skills, and experience as a Software Engineering undergraduate at SLIIT and Full-Stack Developer.

## About This Project

This portfolio website features:

- Responsive design optimized for all device sizes
- Modern UI with smooth animations and transitions
- Interactive sections for projects, contact information, and more
- Clean and professional aesthetic

## Features

- **Hero Section**: Introduces myself with profile image and professional summary
- **Projects Section**: Showcases my work and technical abilities
- **Contact Section**: Easy ways to get in touch with me
- **Responsive Navigation**: Easy navigation on all devices
- **Downloadable Resume**: Direct access to my current CV

## Technologies Used

- React
- Vite
- Tailwind CSS
- Material Design Icons (MDI)
- Lucide React Icons

## Installation

To run this project locally:

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`
